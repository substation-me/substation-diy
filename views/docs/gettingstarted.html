SUBSTATION OVERVIEW ///

 

Substation DIY is a free, open, and secure way to set up simple membership clubs with recurring payments and an easy way to compose and send email to all members.

Let's get it all set up!

 
 

The embed

The widget you see here is a simple embed:

<!-- Substation embed code -->
<script src="https://cdn.jsdelivr.net/gh/substationhq/lodge@0.1/source/lodge.js"></script>
<embed class="lodge" src="{{substationURL}}embed">
 
 

Getting things working.

To get Substation DIY fully up and running you'll need a Braintree account to take payments, and a Mailgun account to send out mail.

Once you have your accounts at Braintree and Mailgun you just need to edit the .env file in your Glitch project. Here it is, step-by-step:

  1. Add an admin email and set the title and main URL (your glitch URL)
    ADMIN_EMAIL=hello@substation.me
    TITLE=Substation
    URL=https://substation.glitch.me/
  2. Set the security secret. This can be anything no one will guess. Mash the keyboard even!
    SECURITY_SECRET=seriouslyanythinglolololsdfjsicuy76Afb
  3. Finally, you need to add information from Braintree and Mailgun. Both need API keys and secrets. Mailgun needs a 'FROM' email address, and Braintree needs you to set up a subscription and add its ID, set the environment as Sandbox or Production, and give a minimum amount for your monthly membership fee.

Customizing this page

You can put the signup widget on any website, but if you want to customize this one (or the email Substation sends) — well you can! Here's how:

  1. Create a folder called /config at the top level of the app
  2. For a custom homepage, create a file called /config/views/index.html — this will be shown instead of the default page
  3. To add CSS create /config/public/custom.css — you can access this file as /customcss from the domain root
  4. You can also customize the email templates —
    • /config/views/email.html for the main email template
    • /config/views/messages/login.html, unsubscribe.html, and welcome.html
      (For these we suggest starting with a copy of the default templates that are located in the main /views folder)

Just a few more things.

  • The admin dashboard is located at /dashboard — login there to see current subscriber stats, get your embed code, export a member list, or send email.
  • Don't worry if the /config folder disappears on you. It's hidden using a .gitignore file so customizations don't get pushed on you. There are two easy fixes. To temporarily un-hide it, just edit the one-line gitignore file so it doesn't match config/ anymore, refresh, then reload. You can change it back once you see the folder. Or you can just delete the .gitignore file completely — the main project needs it, but your remixes maybe don't?
  • Need more than one admin login? Yeah us too. All you need is a little JSON wrapped in $'' in your .env file. You can add any number of admin emails this way, like so:
    ADMIN_EMAIL=$'["hello@substation.me","hi@substation.me"]'
  • Want a nicer looking FROM address for Mailgun? Use the same $'' trick:
    MAILGUN_FROM_EMAIL=$'Substation <hello@substation.me>'
